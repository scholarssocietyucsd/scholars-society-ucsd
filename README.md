# Scholars Society at UCSD Website

A free-hostable static website for Scholars Society at UCSD.

## Run Locally

Open `index.html` in a browser, or serve the folder with any static server:

```sh
python3 -m http.server 8080
```

## Deploy for Free

Good low-cost options:

- GitHub Pages
- Netlify free tier
- Vercel free tier

The site is plain HTML, CSS, and JavaScript, so there is no build step.
